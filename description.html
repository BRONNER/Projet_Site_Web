Types de miels :


L’exploitation réalise les miellées de colza, acacia, châtaignier, montagne, Ardèche, lavande précoce et lavande tardive, sapin, metcalfa, renouée. Les ruches retournent ensuite sur les ruchers d’hivernage dans le sud ou elles peuvent travailler sur la miellée d’arbousier. Les miellées tardives (metcalfa, renouée, arbousier) servent à constituer les provisions des colonies, afin de diminuer les volumes de nourissements. Les associés réalisent de nombreux miels de crus, et peu de miel polyfloraux notamment pour répondre aux besoins spécifiques d’une clientèle exigeante.
